StrataScribe
  • About
Stratagems list for BattleScribe rosters.

How to use this tool

  • 1) Select your battle scribe *.ros or *.rosz file
  • 2) Select options you need
  • 3) Press button and enjoy!

Options

  • Show Units ID in Phase Table
    • Stratagem by Phase table will show, which stratagems can be used with which units. (For Example: VENGEFUL SKIES [1] [2])
    Stratagem by Unit table will have ID for each unit. (For Example: [1] Master of Possession, [2] Chaos Lord in Terminator Armour)
  • Show Phases in Units Table
    • Stratagem by Unit table will have phase name for each stratagem ("Cp" for Command phase, "ESp" for Enemy Shooting phase etc). (For Example: VESSELS FOR THE NEVERBORN [EFp] [Fp])
  • Show no Unit Stratagems
    • Some of Stratagems can be used anytime and don't require any Units for that. If this option is selected - they are added to Stratagem by Phase table.
