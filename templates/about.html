StrataScribe
About
Stratagems list for BattleScribe rosters (powered by Wahapedia)

Repository

https://github.com/ScooloV/StrataScribe

Other BattleScribe beautifiers

  • https://rweyrauch.github.io/PrettyScribe/ - converts your BattleScribe file in printable format
  • https://buttscri.be/ - another tool to convert to printable format
  • https://anti-heresy-sheets.web.app/ - converts your HTML BattleScribe to printable format
  • https://dataslate.rocks/ - another nice BattleScribe roster renderer, which supports not only W40k, but also KT
  • https://40001format.xyz/ - converts your army list to short format for tournament lists

Other Army builders, supporting BattleScribe

  • https://www.forces.app/ - mobile Army builder with different features
  • https://game-datacards.eu/ - here you can create custom cards to print or for mobile use
  • https://40kroller.animetidd.is/ - allows to see not only your BattleScribe army, but much much more
  • https://www.newrecruit.eu/ - army builder which support BattleScribe file format
